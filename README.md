# Library Manager

## Description
- A simple Library Manager web application built with HTML, CSS, and Vanilla JavaScript.
This project allows users to create book objects, update reading status, and remove books from the library dynamically.

## Features
- Add new books to the library
- Store book information as JavaScript objects
- Toggle book status (Read / Not Read)
- Delete books from the library
- UI built with modern CSS
- Dynamic DOM manipulation using Vanilla JavaScript
